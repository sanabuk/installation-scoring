# installation-scoring
POC : établir le scoring d'une installation maraichère bio selon sa position géographique 

Process :
- Sélectionner une ville selon nom ou coordonnées géographiques
- Récupérer les villes dans un rayon de 15km
- Le nombre de fermes maraichères bios
- Pour chacune des communes, récupérer :
    * Le niveau d'imposition sur le revenu
    * La population
    * Le nombre d'exploitations agricoles
    * L'offre touristique d'hébergements
    * Les restrictions d'eau
